# Bundled font sources

The WOFF2 files in `src/fonts` are locally bundled copies of the Latin webfont
builds distributed by [Fontsource](https://fontsource.org/). They are not loaded
from Fontsource or any other third party at runtime. The immutable source URLs
below were downloaded and verified byte-for-byte against the recorded hashes.

- Space Grotesk variable (weight axis):
  `https://cdn.jsdelivr.net/fontsource/fonts/space-grotesk:vf@5.2.10/latin-wght-normal.woff2`
  (`SHA-256 0640890476fc1198ab4de571fb658de443c4d85b66466ec09534a8737ab1ce9d`)
- IBM Plex Mono 400:
  `https://cdn.jsdelivr.net/fontsource/fonts/ibm-plex-mono@5.2.7/latin-400-normal.woff2`
  (`SHA-256 08949f728dc52d528e69b1667d15c89a5686a4ee9a296ff90983985f99c380f7`)
- IBM Plex Mono 500:
  `https://cdn.jsdelivr.net/fontsource/fonts/ibm-plex-mono@5.2.7/latin-500-normal.woff2`
  (`SHA-256 01d285447409c8a588692162439a038b8cbd7871309ee20267b0d2d91c6e8e22`)

The corresponding `*_OFL.txt` files are byte-identical copies of these immutable
upstream Google Fonts license sources and retain the complete SIL Open Font
License notices:

- Space Grotesk OFL:
  `https://raw.githubusercontent.com/google/fonts/03781cf7a714af8431d14b6f337f923c774429d7/ofl/spacegrotesk/OFL.txt`
- IBM Plex Mono OFL:
  `https://raw.githubusercontent.com/google/fonts/03781cf7a714af8431d14b6f337f923c774429d7/ofl/ibmplexmono/OFL.txt`
